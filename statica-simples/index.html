Porque cursar Sistemas da Informação 1?


      Cursar SI1 é uma boa maneira de você, estudante de Ciência da computação, aprender linguagens e técnicas para desenvolvimento Web.


      Neste curso você irá aprender a utilizar:

  • GIT - Controle de versão
  • HTML
  • CSS
  • Bootstrap
  • Javascript (jQuerry)
  • Play! Framework
  • SGBD/JPA
  • Heroku


       O objetivo do curso é que ao final, o aluno esteja capacitado à desenvolver sites de maneira profissional utilizando o que há de mais novo e bem visto no mercado de trabalho.
